# Projek2 Berpikir Komputasi
Membuat program yang dapat mengetahui jumlah gempa yang terjadi di suatu pulau di Indonesia dalam kurun waktu tertentu dan dengan minimal magnitudo yang sudah ditentukan. Kurun waktu, Pulau, dan Min Magnitudo ditentukan oleh pengguna. Data diambil dari usgs earthquake api dengan ketentuan link buat ambilnya yaitu "https://earthquake.usgs.gov/fdsnws/event/1/query?format=geojson&starttime=(diinput user)&endtime=(diinput user)&minmagnitude=(diinput user)&minlatitude=(diinput user)&maxlatitude=(diinput user)&minlongitude=(diinput user)&maxlongitude=(diinput user)".
